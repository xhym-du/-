# 智能瞭望数据分析处理系统

## 项目概述
智能瞭望数据分析处理系统是一个基于Python Flask开发的网络数据抓取与分析平台，能够通过百度搜索引擎获取数据，并提供数据存储、检索和报告生成功能。

## 技术栈
- **后端**：Python 3.8+, Flask
- **前端**：HTML5, CSS3, JavaScript
- **数据库**：SQLite
- **网络请求**：Requests
- **数据解析**：BeautifulSoup4
- **PDF生成**：FPDF
- **虚拟环境**：python-m venv venv

## 项目结构
```
智能瞭望数据分析处理系统/
├── app.py              # Flask应用主文件
├── baidu_spider.py     # 百度爬虫模块
├── venv/               # Python虚拟环境
├── templates/          # HTML模板
│   ├── login.html      # 登录页面
│   ├── dashboard.html  # 后台主页
│   ├── search_results.html  # 搜索结果页面
│   └── data_warehouse.html  # 数据仓库页面
├── static/             # 静态资源目录

└── DEV_LOG.md          # 开发日志
```

## 主要功能

### 1. 用户认证
- 基于SQLite数据库的用户登录系统
- 默认管理员账号：admin / admin888
- 登录状态管理

### 2. 数据抓取
- 百度搜索引擎数据抓取
- 支持自定义搜索关键词
- 动态参数处理

### 3. 数据处理与存储
- 搜索结果实时展示
- 批量数据保存到数据库
- 数据仓库存储原始数据

### 4. 数据检索
- 按关键词搜索数据库内容
- 支持日期归类查看
- 数据筛选功能

### 5. 报告生成
- 将数据导出为PDF格式
- 支持在线预览和下载
- 报告包含完整数据结构

## 安装与运行

### 1. 创建并激活虚拟环境
```bash
# 创建虚拟环境
python -m venv venv

# Windows激活
venv\Scripts\activate

# Linux/Mac激活
source venv/bin/activate
```

### 2. 安装依赖
```bash
pip install flask beautifulsoup4 requests fpdf
```

### 3. 运行应用
```bash
python app.py
```

### 4. 访问系统
在浏览器中输入：http://127.0.0.1:5000

## 使用说明

### 登录系统
1. 访问登录页面
2. 输入用户名：admin
3. 输入密码：admin888
4. 点击登录按钮

### 数据抓取
1. 在后台主页输入搜索关键词
2. 点击搜索按钮获取数据
3. 查看搜索结果列表
4. 选择需要保存的数据并点击"批量保存到数据库"

### 数据管理
1. 点击"数据仓库"进入数据管理页面
2. 使用搜索框查找特定数据
3. 点击"生成PDF报告"创建数据报告
